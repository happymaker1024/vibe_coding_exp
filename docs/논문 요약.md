# 논문 요약

## 1. 논문 개요

| 구분    | 내용                                                                                                                            |
| ----- | ----------------------------------------------------------------------------------------------------------------------------- |
| 논문명   | 설명 가능한 정기예금 가입 여부 예측을 위한 앙상블 학습 기반 분류 모델들의 비교 분석                                                                              |
| 영문명   | A Comparative Analysis of Ensemble Learning-Based Classification Models for Explainable Term Deposit Subscription Forecasting |
| 연구 목적 | 은행 고객의 정기예금 가입 여부를 예측하고, 예측 근거를 SHAP으로 설명하는 것                                                                                 |
| 데이터   | Kaggle Bank Marketing Dataset                                                                                                 |
| 주요 기법 | RF, GBM, XGBoost, LightGBM, SHAP                                                                                              |
| 평가 방식 | 10-Fold Cross-Validation                                                                                                      |
| 평가 지표 | Accuracy, Kappa, F1-Score                                                                                                     |

이 논문은 단순히 정기예금 가입 여부를 잘 맞히는 모델을 만드는 데서 끝나지 않고, **왜 해당 고객을 가입 또는 미가입으로 예측했는지 설명 가능한 구조**를 만드는 데 초점을 둔다. 즉, “성능 좋은 분류 모델 + SHAP 기반 해석”이 핵심 구조다. 

---

## 2. 연구 문제

기존 연구에서는 정기예금 가입 여부 예측에 ANN, DNN, SVM, 의사결정나무 등의 모델을 사용했지만, 예측 결과에 대한 설명력이 부족하다는 한계가 있었다. 금융 분야에서는 모델의 정확도뿐 아니라 **고객 정보 중 어떤 변수가 예측에 영향을 미쳤는지**, **특정 고객이 왜 가입 가능성이 높거나 낮게 판단되었는지**를 설명할 수 있어야 한다.

따라서 이 논문의 연구 문제는 다음과 같이 정리할 수 있다.

> 정기예금 가입 여부 예측에서 트리 기반 앙상블 모델 중 어떤 모델이 가장 우수하며, SHAP을 활용해 예측 근거를 설명할 수 있는가?

---

## 3. 데이터와 전처리

논문은 Kaggle의 **Bank Marketing Dataset**을 사용한다. 데이터는 포르투갈 은행의 텔레마케팅 고객 정보를 기반으로 하며, 총 **11,163개 관측치**로 구성되어 있다. 종속변수는 `deposit`, 즉 정기예금 가입 여부다. 

주요 변수는 다음과 같다.

| 변수 유형     | 예시                                      |
| --------- | --------------------------------------- |
| 고객 인적 정보  | age, job, marital, education            |
| 금융 정보     | balance, default, housing, loan         |
| 마케팅 접촉 정보 | contact, day, month, duration, campaign |
| 이전 캠페인 정보 | pdays, previous, poutcome               |
| 종속변수      | deposit                                 |

전처리는 비교적 단순하다.

| 전처리 항목         | 처리 방식            |
| -------------- | ---------------- |
| 문자형 변수         | One-Hot Encoding |
| yes/no 변수      | yes=1, no=0      |
| 종속변수 `deposit` | 가입=1, 미가입=0      |
| 최종 독립변수 수      | 48개              |

재현할 때 중요한 점은 **`drop_first=True`를 쓰지 않는 것**이다. 논문은 원-핫 인코딩 후 총 48개 독립변수를 사용했다고 설명한다. 따라서 더미변수를 하나씩 제거하면 논문과 변수 수가 달라질 수 있다. 

---

## 4. 실험 설계

논문은 총 4개의 트리 기반 앙상블 모델을 비교한다.

| 계열  | 모델            |
| --- | ------------- |
| 배깅  | Random Forest |
| 부스팅 | GBM           |
| 부스팅 | XGBoost       |
| 부스팅 | LightGBM      |

실험은 **10겹 교차검증**으로 수행된다. 전체 데이터를 10개 부분으로 나누고, 매번 9개는 학습용, 1개는 검증용으로 사용한다. 이 과정을 10번 반복해 평균 성능을 계산한다.

평가 지표는 다음 세 가지다.

| 지표       | 의미                      |
| -------- | ----------------------- |
| Accuracy | 전체 예측 중 맞춘 비율           |
| Kappa    | 우연 일치를 보정한 일치도          |
| F1-Score | Precision과 Recall의 조화평균 |

논문 실험 환경은 Python 3.7.9, scikit-learn 0.24.2, xgboost 1.4.2, lightgbm 3.2.1이다. 모든 모델의 `random_state`는 42로 설정하였다. 

---

## 5. 핵심 실험 결과

논문은 두 가지 데이터 구성으로 실험한다.

## 5.1 Duration 포함 실험

`duration`은 마지막 통화시간이다. 이 변수를 포함하면 모델 성능이 크게 높아진다.

| 모델       |  Accuracy |     Kappa |  F1-Score |
| -------- | --------: | --------: | --------: |
| RF       |     0.857 |     0.715 |     0.850 |
| GBM      |     0.863 |     0.726 |     0.860 |
| XGBoost  |     0.855 |     0.710 |     0.851 |
| LightGBM | **0.864** | **0.727** | **0.861** |

결과적으로 **LightGBM이 가장 우수한 성능**을 보였다. 다만 GBM과의 차이는 매우 작다. 

---

## 5.2 Duration 제외 실험

논문은 `duration` 변수를 별도로 제거한 실험도 수행한다. 이유는 실제 금융기관이 잠재 고객을 사전에 예측할 때, 통화시간은 아직 알 수 없는 정보이기 때문이다. 즉, `duration`은 예측 시점에서는 사용하기 어려운 변수다.

| 모델       |  Accuracy |     Kappa |  F1-Score |
| -------- | --------: | --------: | --------: |
| RF       |     0.729 |     0.452 | **0.694** |
| GBM      | **0.738** | **0.469** |     0.693 |
| XGBoost  |     0.729 |     0.453 |     0.690 |
| LightGBM |     0.737 |     0.466 |     0.690 |

`duration`을 제외하면 전체 성능이 크게 하락한다. 이 조건에서는 **GBM이 Accuracy와 Kappa 기준으로 가장 우수**했다. 다만 F1-Score는 RF가 0.694로 가장 높다.

## 6. SHAP 해석 결과

논문은 최종적으로 성능이 가장 좋은 모델에 SHAP을 적용해 변수 영향도를 분석한다.

### Duration 포함 데이터

가장 중요한 변수는 `duration`이었다. 즉, 마지막 통화시간이 정기예금 가입 여부 예측에 가장 큰 영향을 주었다. SHAP Summary Plot에서는 `duration`, `contact_unknown`, `poutcome_success`, `housing`, `month_may`, `age`, `balance` 등이 주요 변수로 나타난다. 

논문 결론에서는 다음과 같은 해석을 제시한다.

| 변수       | 해석                          |
| -------- | --------------------------- |
| Duration | 통화시간이 주요 영향 변수              |
| Age      | 30대~50대 고객이 정기예금 가입에 긍정적 반응 |
| Housing  | 주택담보대출 보유 여부가 예측에 영향        |
| Loan     | 개인대출 보유 여부가 예측에 영향          |

특히 논문은 **통화시간이 500초보다 짧을 때**, **주택담보대출 또는 개인대출을 보유할 때**, **30대부터 50대까지의 고객**이 정기예금 가입에 긍정적인 반응을 보였다고 해석한다. 

### Duration 제외 데이터

`duration`을 제외하면 특정 변수 하나가 압도적으로 지배하기보다 여러 변수가 함께 예측에 작용한다. 논문은 오히려 `duration`을 제외한 데이터셋이 여러 독립변수를 효과적으로 고려하여 예측값을 도출한다고 해석한다.

## 7. 논문의 핵심 기여

이 논문의 기여는 세 가지로 정리할 수 있다.

| 기여        | 설명                                                |
| --------- | ------------------------------------------------- |
| 모델 비교     | RF, GBM, XGBoost, LightGBM을 동일 데이터와 동일 평가 방식으로 비교 |
| 설명가능성 확보  | SHAP을 사용해 변수 영향도와 개별 예측 근거 제시                     |
| 현실적 변수 검토 | `duration` 포함/제외 실험을 통해 실제 예측 상황을 고려              |

특히 `duration` 변수를 제외한 실험은 중요하다. 단순 성능만 보면 `duration` 포함 모델이 더 좋지만, 실제 사전 마케팅 예측에서는 통화시간을 알 수 없기 때문에 `duration` 제외 실험이 더 현실적이다.

## 8. 재현 관점에서 중요한 포인트

`bank_marketing_data/` 폴더에 데이터가 이미 있다면, 재현은 다음 순서로 진행하면 된다.

```text
1. 데이터 로드
2. deposit을 target으로 분리
3. yes/no 변수 1/0 변환
4. 문자형 변수 원-핫 인코딩
5. 전체 변수 데이터셋 생성
6. duration 제거 데이터셋 생성
7. RF, GBM, XGBoost, LightGBM 학습
8. 10-Fold Cross-Validation 수행
9. Accuracy, Kappa, F1-Score 계산
10. LightGBM 또는 GBM에 SHAP 적용
```

재현에서 가장 중요한 검증 기준은 다음이다.

| 확인 항목       | 기준                                      |
| ----------- | --------------------------------------- |
| 데이터 크기      | 약 11,163행                               |
| 전처리 후 변수 수  | 48개 독립변수에 가까워야 함                        |
| Duration 포함 | LightGBM이 약 0.864 Accuracy 근처           |
| Duration 제외 | GBM/LightGBM이 약 0.737~0.738 Accuracy 근처 |
| SHAP 주요 변수  | Duration 포함 시 `duration`이 최상위           |

---

# 한 줄 결론

이 논문은 **Bank Marketing Dataset을 대상으로 RF, GBM, XGBoost, LightGBM을 비교하고, 가장 우수한 모델에 SHAP을 적용해 정기예금 가입 예측의 근거를 설명한 실험 논문**이다. 재현 실습에서는 **Duration 포함/제외 실험을 분리하는 것**이 가장 중요하다.
